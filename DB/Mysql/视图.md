### 导入导出权限和服务器ip问题

视图是一个虚拟表，其内容由查询定义。
同真实的表一样，视图包含一系列带有名称的列和行数据。
视图的数据变化会影响到基表，基表的数据变化也会影响到视图。

CREATE ALGORITHM = UNDEFINED DEFINER = `php` @`172.16.100.%` SQL SECURITY DEFINER VIEW `aaa` AS SELECT
`xx`.`domain_id` AS `domain_id`,
`xx`.`package_id` AS `package_id` 
FROM
	( `a` JOIN `b` ) 
WHERE
	( `a`.`domain_id` = `b`.`id` )
	
	
表要占用磁盘空间，视图不需要 
视图不能添加索引 
使用视图可以简化查询
视图可以提高安全性
