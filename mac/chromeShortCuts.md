## 访问 DevTools

| 访问 DevTools	                     |      在 Windows 上	                      |  在 Mac 上         |
|------------------------------------|:----------------------------------------- :|-------------------:|
| 打开 Developer Tools	             |  打开 Developer Tools	F12、Ctrl + Shift + I | Cmd + Opt + I      |
| 打开/切换检查元素模式和浏览器窗口	     |    Ctrl + Shift + C	                      |   Cmd + Shift + C  |
| 打开 Developer Tools 并聚焦到控制台	 | Ctrl + Shift + J	                          |    Cmd + Opt + J   |
